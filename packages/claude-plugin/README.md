# @codelumeai/claude-plugin

The CodeLumeAI Claude Code plugin. Adds the `/codelumeai` slash-command family to any Claude Code surface (CLI, VS Code, JetBrains, desktop, web).

The full spec lives in [`SKILL.md`](./SKILL.md). The system prompts live in [`prompts/`](./prompts).
