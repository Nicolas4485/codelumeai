# Pick a default mode

CodeLumeAI has three display modes. Pick the one that fits how you read code.

## The three modes

- **`hover`** _(default)_ — Translation tooltips appear on demand when you mouse over a line. Cleanest, lowest visual noise.
- **`always-on`** — A short summary appears at the end of every chunk in the editor, all the time. Best for studying a file.
- **`off`** — Disabled. The extension still loads but doesn't render anything.

## Switching modes

There's a status-bar item in the **bottom-right** of the window: `👁 CodeLumeAI: hover`. **Click it** to cycle Off → Hover → Always-on.

You can also change the default in VS Code Settings — search for `codelumeai.mode`.
